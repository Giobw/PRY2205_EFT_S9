# Evaluación Final Transversal (EFT) - Semana 9 

**Asignatura:** PRY2205 - Consulta de Bases de Datos

## Descripción del Proyecto 
Este repositorio contiene el script SQL correspondiente a la Evaluación Final Transversal. El código demuestra la aplicación de los siguientes conceptos:
* Creación y gestión de Usuarios, Roles y Privilegios en Oracle SQL.
* Implementación de seguridad mediante Sinónimos Públicos.
* Creación de Vistas complejas y Tablas de reporte aplicando reglas de negocio (reajustes matemáticos).
* Optimización de consultas mediante la creación de Índices.

## Archivos 
* `EFT_S9_Giovanni_Bencini.sql`: Script consolidado con la solución completa, organizada secuencialmente por usuario (Admin, Dueño, Desarrollador y Consultor).
